# 12149 — Feynman

## 📘 題目敘述

輸入多個整數 `N`（`1 ≤ N ≤ 100`），代表 `N×N` 方格。對每個 `N`，輸出該網格內所有大小的正方形總數。輸入以 `0` 結束且不處理。

## 💡 解題提示（Hint）

* 大小為 `k×k` 的正方形有 `(N−k+1)^2` 個。
* 總數：$\sum_{k=1}^{N} (N-k+1)^2=\sum_{i=1}^{N} i^2=\frac{N(N+1)(2N+1)}{6}$。
* 直接套用公式即可（`N=100` 時結果 338350，整數安全）。

## Sample input:

```
2
1
8
0
```

## Sample Output:

```
5
1
204
```
